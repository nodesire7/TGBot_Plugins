# TGBot Plugin Marketplace

Official plugin marketplace for [TGBot Admin](https://github.com/nodesire7/TGBot_Admin). Browse, install, and manage plugins to extend your Telegram bot's functionality.

## Quick Start

### Installing Plugins

1. Browse available plugins in the [plugins/](./plugins/) directory
2. Each plugin folder contains installation instructions in its README.md
3. Use the TGBot Admin dashboard to install plugins directly from this marketplace

**Via API:**
```bash
# Install a plugin from the marketplace
curl -X POST http://localhost:8000/api/market/install/arithmetic_verification
```

**Via Web UI:**
1. Navigate to Plugins → Marketplace
2. Browse or search for plugins
3. Click "Install" on desired plugin

## Plugin Standards

All plugins follow strict standards for consistency and security:

- **Versioning**: [Semantic Versioning 2.0.0](https://semver.org/)
- **Documentation**: Each plugin includes comprehensive README
- **Configuration**: JSON Schema for all configurable options
- **Security**: Plugins declare required permissions explicitly

See [Plugin Standards](./docs/STANDARDS.md) for complete specifications.

## Directory Structure

```
plugins/
├── your_plugin/
│   ├── manifest.json      # Required: Plugin metadata
│   ├── main.py            # Required: Entry point
│   ├── README.md          # Required: Documentation
│   ├── config_schema.json # Optional: Configuration schema
│   ├── requirements.txt   # Optional: Python dependencies
│   └── tests/             # Optional: Unit tests
```

## Third-Party Repositories

TGBot Admin supports third-party plugin repositories. To add a custom repository:

```bash
curl -X POST http://localhost:8000/api/market/repositories \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://raw.githubusercontent.com/user/my-plugins/main",
    "name": "My Custom Plugins",
    "branch": "main"
  }'
```

Your repository must have:
- `index.json` at root (same format as this repo)
- Each plugin in its own directory with `manifest.json` and `main.py`

## Plugin Index
